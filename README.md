# Ruby on Rails Study 📚

루비 언어 기초부터 레일즈 웹 프레임워크, 그리고 Vue.js 연동 프로젝트까지 학습한 기록을 모아둔 저장소입니다.

## 📂 디렉토리 설명

### 1. `ruby_programming_language/`
* **루비 언어 기초 학습**
* 변수, 클래스, 블록, 이터레이터 등 루비 문법을 익히기 위해 작성한 스크립트 파일들입니다.

### 2. `my_first_app/`
* **레일즈 기초 프로젝트**
* Rails의 동작 원리(MVC 패턴)를 이해하고, Scaffold를 활용해 기본적인 게시판(CRUD) 기능을 구현해 본 연습용 앱입니다.

### 3. `Simchat/`
* **실시간 채팅 애플리케이션 (Vue + Rails API)**
* **simchat-api**: Rails 8 (API 모드) + ActionCable을 활용한 백엔드.
* **simchat-client**: Vue 3 + Pinia를 활용한 프론트엔드.
* 웹소켓 통신을 이용해 실시간으로 대화를 주고받는 기능을 구현했습니다.
